Cайт для сбора отзывов о разных компаниях. 
Основные страницы:

Страница 1. Оставить отзыв о компании.
На этой странице пользователю предоставляется возможность оставить отзыв, заполнив следующие данные:
•	Имя пользователя *;
•	Название организации *;
•	Её адрес;
•	Что понравилось *;
•	Что не понравилось;
•	Комментарии;
•	Оценка 1..5 *.

Страница 2. Список отзывов.
На этой странице пользователи могут почитать отзывы других пользователей. 
За раз подгружается 5 отзывов. Загрузка новых отзывов происходит без рефреша страницы.

Страница 3. Подробности по отзыву.
На этой странице можно почитать, что понравилось, что не понравилось, а также комментарии. 

Страница 4. Раздел для администраторов.
На этой странице можно также просматривать список отзывов, как и на странице 2.
Но помимо этого на странице можно удалить любой отзыв. 
Переход на эту страницу возможен только по прямой ссылке "http://.../admin".
